## Project Overview
This project performs end-to-end business analysis on customer churn data
to identify factors influencing customer retention and provide
data-driven business recommendations.

## Dataset
- customer_churn.csv (500+ records)

## Analysis Performed
- Data cleaning and preparation
- Exploratory Data Analysis (EDA)
- Churn distribution analysis
- Correlation analysis
- Hypothesis testing (t-test)
- Business insights and recommendations

## Key Finding
Monthly charges have a statistically significant impact on customer churn
(p-value = 0.016).
